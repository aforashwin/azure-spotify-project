# azure-spotify-project
End-to-end Azure Data Engineering project using Azure Data Factory, Azure SQL Database, ADLS Gen2, Azure Databricks, Unity Catalog, PySpark, Auto Loader, Delta Live Tables, and Delta Lake. Implements incremental ETL, Medallion Architecture, streaming, and snowflake Schema modeling.
